- 覆盖内建函数

  #+BEGIN_SRC python
    abs = 10
  #+END_SRC

  abs 函数实际上是定义在 builtins 模块中的，所以要想在其它模块中也生效，要用：

  #+BEGIN_SRC python
    import builtins; builtins.abs = 10
  #+END_SRC
