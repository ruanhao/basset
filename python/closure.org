** 返回闭包时注意：返回函数不要引用任何循环变量，或者后续会发生变化的变量

#+BEGIN_SRC python
def count():
    fs = []
    for i in range(1, 4):
        def f():
             return i*i
        fs.append(f)
    return fs

f1, f2, f3 = count()
#+END_SRC

#+BEGIN_SRC bash
>>> f1()
9
>>> f2()
9
>>> f3()
9
#+END_SRC

如果一定要引用循环变量则再创建一个函数，用该函数的参数绑定循环变量当前的值，无论该循环变量后续如何更改，已绑定到函数参数的值不变：

#+BEGIN_SRC python
def count():
    def f(j):
        def g():
            return j*j
        return g
    fs = []
    for i in range(1, 4):
        fs.append(f(i)) # f(i)立刻被执行，因此i的当前值被传入f()
    return fs
#+END_SRC
