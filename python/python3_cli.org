* 1，virtualenv

*虚拟环境* 是系统的一个位置，可以在其中安装包，并将其与其他 Python 包隔离，将项目的库与其他项目分离是有益的。

#+BEGIN_SRC sh
  virtualenv <name> --python=python3
  source <name>/bin/activate
  deactivate                      # 停止使用虚拟环境
#+END_SRC
