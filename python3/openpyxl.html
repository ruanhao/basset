1.4 获取单元格

1.4.1 使用字符串坐标

根据名字访问 Cell 对象： 

cell_a1 = active_sheet['A1']
log("cell_a1", cell_a1)
log("cell_a1.value", cell_a1.value)
log("cell_a1.row", cell_a1.row)
log("cell_a1.column", cell_a1.column)
log("cell_a1.coordinate", cell_a1.coordinate)

========== cell_a1 ===========
<Cell 'Sheet1'.A1>
======= cell_a1.value ========
2015-04-05 13:34:02
======== cell_a1.row =========
1
======= cell_a1.column =======
A
===== cell_a1.coordinate =====
A1

1.4.2 使用整数坐标

第一行或第一列索引是 1 不是 0 

c = active_sheet.cell(row=1, column=1)
log("c.value", c.value)

========== c.value ===========
2015-04-05 13:34:02

1.4.3 列字母和数字之间的转换

from openpyxl.utils import get_column_letter, column_index_from_string
from openpyxl.utils import coordinate_from_string, column_index_from_string

log("get_column_letter(1)", get_column_letter(1))

xy = coordinate_from_string('A4') # returns ('A',4)
row = xy[1]
col = column_index_from_string(xy[0]) # returns 1
log("xy", xy)
log("row", row)
log("col", col)

==== get_column_letter(1) ====
A
============= xy =============
('A', 4)
============ row =============
4
============ col =============
1

2 写入 Excel 文档

2.1 创建并保存

当修改 Workbook 对象或它的工作表和单元格时，电子表格文件不会保存，需要调用 save() 方法。 

sheet = wb.active
sheet.title = 'Spam'
wb.save('/tmp/example_copy.xlsx')

2.2 创建和删除工作表

wb.create_sheet()
wb.create_sheet(index=0, title='First')
wb.create_sheet(index=2, title='Last')
log("wb.sheetnames", wb.sheetnames)
wb.remove(wb['Last'])
wb.save('/tmp/example_create_delete_sheet.xlsx')

======= wb.sheetnames ========
['First', 'Spam', 'Last', 'Sheet2', 'Sheet3', 'Sheet']

2.3 写入单元格

sheet['A1'] = 'Hello world'
log("sheet['A1'].value", sheet['A1'].value)

===== sheet['A1'].value ======
Hello world

2.4 公式

公式以一个等号开始。 

sheet['B9'] = '=SUM(B1:B8)'
