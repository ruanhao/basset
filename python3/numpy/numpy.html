1.1.4 整形数组 (reshape)

reshape 并不创建新的数组，而是返回一个 视图 。 

origin = np.arange(9)
reshaped = origin.reshape(3, 3)
log("reshape 3*3", reshaped)
log("origin (unchaged)", origin)
reshaped[0,1] = 100
log("origin (also changed)", origin)

======== reshape 3*3 =========
[[0 1 2]
 [3 4 5]
 [6 7 8]]
===== origin (unchaged) ======
[0 1 2 3 4 5 6 7 8]
=== origin (also changed) ====
[  0 100   2   3   4   5   6   7   8]

1.2.3 数组索引

data = np.arange(16).reshape(4,4)
log("data", data)

xidx = [0, 2, 3]
yidx = [1, 2, 3]
log("data[[0,2,3],[1,2,3]]", data[xidx, yidx])

# 修改每行里的特定列的数据
yidx = [0, 3, 2, 0]
data[np.arange(4), yidx] += 100
log("data", data)

1.2.4 布尔索引

data = np.arange(16).reshape(4,4)

idx = data >= 10

log("data[data%2 == 0]", data[data%2 == 0])

===== data[data%2 == 0] ======
[ 0  2  4  6  8 10 12 14]

1.3 运算

x = np.arange(1,5).reshape(2,2)
y = np.arange(5,9).reshape(2,2)
log("x", x)
log("y", y)
log("x + y", x + y)
log("np.add(x, y)", np.add(x, y))
log("y - x", y - x)
log("np.subtract(y, x)", np.subtract(y, x))
log("x * y", x * y)
log("x / y", x / y)
x = np.array(x, dtype=float)
y = np.array(y, dtype=float)
log("x", x)
log("y", y)
log("x / y", x / y)
log("np.sqrt(x)", np.sqrt(x))
log("x + 10", x + 10)

data = np.arange(16).reshape(4,4)
log("data.sum()", data.sum())
log("data.sum(axis=0)", data.sum(axis=0))
