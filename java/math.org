* BigDecimal

比较好的 round 方法

#+BEGIN_SRC java
  class MyMath {
      public static double round(double num, int scale) {
          BigDecimal big = new BigDecimal(num);
          BigDecimal result = big.divide(new BigDecimal(1), scale, BigDecimal.ROUND_HALF_UP);
          return result.doubleValue();
      }
  }
#+END_SRC
