* ByteArrayOutputStream

可以将字节流转换为数组

#+BEGIN_SRC java
ByteArrayOutputStream s = new ByteArrayOutputStream();
s.write(1);
s.write(2);
s.write(3);
byte[] bytes = s.toByteArray();
#+END_SRC
