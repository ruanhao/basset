* 1, 基本原理

离开当前指定类型的 namespace ，创建且加入新的 namespace ，然后执行参数中指定的命令

* 2, 常用命令

#+BEGIN_SRC
# 创建新 IPC namespace
unshare --ipc /bin/bash

# 创建新 UTS namespace
unshare --uts /bin/bash

# 创建新 mount namespace
unshare --mount /bin/bash

# 创建新 pid namespace
unshare --pid --fork --mount-proc /bin/bash
#+END_SRC
