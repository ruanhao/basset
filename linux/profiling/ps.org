* 1, 常用命令

- 根据字段排序

#+BEGIN_SRC sh
  ps -eo pid,ppid,cmd,%mem,%cpu --sort=-%mem
  ps -eo pid,ppid,cmd,pmem,pcpu --sort=-pmem
#+END_SRC


- 查看特定进程的线程信息

  =ps -L <pid>=

- 查看启动时间和自启动后至现在的时间

  =ps -eo pid,lstart,etime,cmd=

  #+BEGIN_EXAMPLE
    lstart: started time when the command started
    etime:  elapsed time since the process was started
  #+END_EXAMPLE
