#+TITLE: GNU find
#+OPTIONS: ^:nil

* Find by time
** Find files modified less than 10 minutes ago
=find / -mmin -10=
** Find files modified less than 1 hour
=find / -mtime -1=
** Find files accessed (read) less then 1 minute
=find . -amin -1=

* Find by filesystem
** Don't find other filesystem
=find / -name "*.txt" -mount=
=find / -name "*.txt" -xdev=
** Specify a certain filesystem
=find / -name "*.txt" -fstype vfat=

* Find by size
** By byte
=find . -size +1024c=
** By 512 bytes (block)
=find . -size +2=
** By 1024 byte
=find . -size +1k=
** Find empty file
=find . -size 0=
** Find empty directory
=find . -empty=

* Find by directory
** Specify multiple directories
=find /usr /home /tmp -name \*.jar=
