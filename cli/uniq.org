* 文件并集，

  #+BEGIN_SRC sh
    cat a b | sort | uniq > c        # c is a union b
  #+END_SRC


* 文件交集

#+BEGIN_SRC sh
    cat a b | sort | uniq -d > c     # c is a intersect b
#+END_SRC


* 文件差集

#+BEGIN_SRC sh
    cat a b b | sort | uniq -u > c   # c is set difference a - b
#+END_SRC
